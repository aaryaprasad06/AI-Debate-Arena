# AI Debate Arena ⚖️

An AI-powered debate platform where:

- Users enter a debate topic
- A Pro AI agent argues in favor
- An Against AI agent argues against
- An AI Judge evaluates the debate
- Results are displayed in an interactive Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- GitHub
- Gemini API (planned)

## Features

- Multi-agent debate simulation
- Judge verdict system
- Debate history
- Interactive UI
- Future PDF export support

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
